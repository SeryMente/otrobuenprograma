# Rediseno del portafolio - molde Long Now

Este documento rige la reestructura del instrumento tomando como molde la
arquitectura de The Long Now Foundation (longnow.org): un sitio de renombre
que presenta varios proyectos bajo una sola filosofia, con la captacion de
fondos como una capa subordinada y no como el eje.

No se clona su contenido, sus imagenes ni su CSS (son suyos). Se clona su
ORDEN: como ordena la vision, los proyectos, las ideas, la comunidad, las
personas y, al final, el donativo.

## Mapa Long Now -> instrumento

| Long Now      | Nuestra seccion                         |
|---------------|-----------------------------------------|
| Vision        | hero #inicio + #porque / #fundamentos   |
| Projects      | #proyectos (las insignias)              |
| Ideas / Talks | #camino (linea de tiempo) + #pulso      |
| The Interval  | #sumarse + #voces + #mapa + #gracias    |
| People        | #autor + #fundamentos                   |
| Donate        | #cuentas (capa subordinada)             |

## Avance parte por parte

Cada parte entrega un zip nuevo y un conteo de elementos terminados. El
usuario revisa y se avanza a la siguiente.

- [x] Parte 1/6 - Vision (hero + tesis + plomeria de version)
- [ ] Parte 2/6 - Projects (#proyectos: insignias; tercer proyecto por definir)
- [ ] Parte 3/6 - Ideas / Talks (#camino + #pulso)
- [ ] Parte 4/6 - The Interval (#sumarse + #voces + #mapa + #gracias)
- [ ] Parte 5/6 - People (#autor + #fundamentos)
- [ ] Parte 6/6 - Donate (#cuentas multiproyecto)

## No-regresion

Ninguna parte elimina capas ya vivas (navigator, build-tag, temas, glosa,
cuentas claras, timeline, presencia, PWA). Solo reordena y reviste.
